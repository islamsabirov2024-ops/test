# Kino Bot Template

Bu template asosiy bot ichidan token kiritilganda ishga tushadigan kino bot uchun.
Hozirgi real handlerlar `app/main.py` ichida bitta joyda jamlangan, lekin template funksiyalari quyidagilarni qamrab oladi:

- kino kod orqali yuborish
- serial/qism tizimi
- premium tariflar
- chek yuborish va admin tasdiqlashi
- majburiy obuna
- reklama preview va rejalash
- qidiruv, TOP, yangi kinolar, sevimlilar
- anti-spam
