## Ishlash sxemasi
- User glavni botga kiradi va bot turini tanlaydi: Kino bot yoki Reklama tozalovchi bot.
- User BotFatherdan token olib yuboradi.
- Token bazaga saqlanadi, lekin bot to'lov tasdiqlanmaguncha ishlamaydi.
- User chek yuboradi.
- Super admin chekni tasdiqlaydi.
- Super admin botni yoqadi/o'chiradi.
- Har bir bot egasi o'z botida admin bo'ladi.
