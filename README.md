# Aiogram MultiBot + Kino Bot FINAL PRO

## Ishga tushirish

1. `.env.example` faylini `.env` qilib ko'chiring.
2. `BOT_TOKEN` va `SUPER_ADMIN_ID` yozing.
3. Railway/Render yoki lokalda ishga tushiring:

```bash
pip install -r requirements.txt
python -m app.main
```
